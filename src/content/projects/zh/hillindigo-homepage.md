Hillindigo 是我为自己建立的长期数字花园。它不是一个只展示个人简介的名片，而是一套可以持续写作、检索、归档和迭代的内容系统。

## 为什么选择静态生成

个人博客的大多数页面是读多写少。Astro 在构建阶段生成 HTML，访问时不需要持续运行服务器，适合托管在 GitHub Pages，也降低了长期维护成本。

## 当前能力

- 中文与英文独立路由
- Markdown / MDX 内容集合
- 全文搜索、RSS、标签与阅读时间
- 自动文章目录与代码高亮
- 可配置评论、访问统计与图片灯箱

接下来会逐步补充真实内容、项目案例和生活影像，并持续改善可访问性与加载性能。
