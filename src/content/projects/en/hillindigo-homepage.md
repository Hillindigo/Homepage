Hillindigo is a long-term digital garden rather than a profile card. It is designed as a system for writing, searching, archiving and revisiting ideas over time.

## Why static generation

Most personal publishing is read-heavy and write-light. Astro produces HTML during the build, so the site needs no always-on application server and can live comfortably on GitHub Pages.

## Current capabilities

- Separate Chinese and English routes
- Markdown and MDX content collections
- Full-text search, RSS, tags and reading time
- Generated table of contents and syntax highlighting
- Configurable comments, analytics and image lightbox

## What comes next

The next iterations will add deeper writing, more complete project cases and real life imagery while improving accessibility and performance.
